# COVID-19 Analyzer
## Usage:
### Download the app  

### Open it in Terminal (or command line)  

### Create virtual environment:
``` bash
python3 -m venv venv
```
### Activate virtual environment
Windows:
``` bash
venv\Scripts\activate
```
Mac/Linux:
``` bash
source venv/bin/activate
```
### Install modules:
``` bash
pip3 install -r requirements.txt
```
### Run the app
``` bash
python3 main.py 
```
